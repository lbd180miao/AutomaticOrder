## 📋 项目概览

### 目标
将DM 3D深度相机SDK集成到Django项目中，提供REST API和Web界面，用于控制相机、采集深度数据和管理配置。

### 完成时间
2024年（集成开发完成）

### 技术栈
- **后端框架**: Django 5.2+
- **SDK绑定**: Python ctypes → C DLL
- **数据存储**: SQLite
- **API风格**: REST (JSON)
- **前端技术**: HTML5 + JavaScript (原生)
- **数据处理**: numpy, Pillow
