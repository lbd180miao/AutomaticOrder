## 🔄 工作流程

### 场景1：标定并保存配方
1. 用户打开泡棉检测工作台
2. 选择 POS（例如 POS 0）
3. 刷新预览获取相机画面
4. 选择"左侧泡棉"，拖拽框选左侧 ROI
5. 选择"右侧泡棉"，拖拽框选右侧 ROI
6. 点击"💾 保存到配方"按钮
7. 选择目标 POS（可以保存到任意 POS）
8. 配方保存成功，显示提示信息

### 场景2：使用配方检测
1. 用户打开工作台
2. 选择 POS 1
3. 页面自动加载 POS 1 的配方
4. 配方信息卡片显示配方详情
5. 预览图自动显示配方的 ROI 位置
6. 点击"📸 拍照检测"
7. 系统自动使用 POS 1 配方参数
8. 检测结果显示"第2层泡棉检测配方"

### 场景3：临时切换配方
1. 当前 POS 是 0
2. 点击"临时使用其他配方"
3. 选择 POS 2 的配方
4. 卡片变为橙色背景（临时状态）
5. 点击检测，使用 POS 2 配方
6. 检测完成后自动恢复 POS 0 配方
