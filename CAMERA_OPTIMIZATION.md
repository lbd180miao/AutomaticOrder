# 海康相机图像获取优化说明

## 优化内容

### 1. **改进资源管理**
- 修复相机资源泄漏问题：确保相机在捕获失败时正确关闭
- 优化关闭逻辑：成功捕获后不关闭相机（提高重复使用效率），仅在失败时关闭
- 添加 `camera_opened` 标志追踪相机状态

### 2. **增强错误处理**
- **更详细的错误信息**：
  - 超时错误：提示检查相机通电、网络连接和IP配置
  - 设备未找到：提示检查相机连接、驱动安装和SDK工作状态
  - 权限错误：区分 PermissionError 和 OSError
  
- **改进导入错误提示**：
  ```python
  无法导入 chg_hik 模块。请确保：
  1. 已安装海康威视 MVS SDK
  2. 已编译安装 chg_hik Python 绑定 (maturin develop --release)
  3. SDK 的 Env.json 路径配置正确
  ```

### 3. **增加超时保护**
- 添加 `subprocess.TimeoutExpired` 异常处理
- 明确提示 30秒超时错误，方便快速定位问题

### 4. **文件验证增强** (`camera.py`)
- 验证返回的图像路径不为空
- 验证图像文件实际存在
- 验证图像文件大小不为0（避免空文件）

### 5. **目录权限检查优化**
- 区分不同类型的目录错误：
  - `PermissionError`: 权限不足
  - `OSError`: 操作系统错误
  - 其他异常：通用错误
- 提供更具体的错误消息

## 常见问题解决

### 问题1: 相机连接超时
**错误消息**: "相机连接超时"
**解决方案**:
1. 检查相机是否通电
2. 验证网络连接是否正常
3. 确认IP地址配置正确（CAMERA_IP 和 PC_IP）

### 问题2: 未找到相机设备
**错误消息**: "未找到相机设备"
**解决方案**:
1. 确认相机已正确连接
2. 检查相机驱动是否安装
3. 验证 MVS SDK 是否正常工作

### 问题3: 输出目录权限不足
**错误消息**: "输出目录权限不足"
**解决方案**:
1. 检查目录权限设置
2. 确保运行用户有写入权限
3. 在 Windows 上检查目录是否被其他程序锁定

### 问题4: 无法导入 chg_hik
**错误消息**: "无法导入 chg_hik 模块"
**解决方案**:
1. 安装海康威视 MVS SDK
2. 编译并安装 Python 绑定: `maturin develop --release`
3. 检查 SDK 的 Env.json 路径配置
