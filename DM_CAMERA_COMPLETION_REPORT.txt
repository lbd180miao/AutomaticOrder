三、技术实现
================================================================================

【架构层次】

1. SDK包装层 (sdk_wrapper.py)
   - ctypes加载C DLL
   - 封装所有SDK函数
   - 异常处理和类型转换
   代码量: ~500行

2. 业务逻辑层 (services.py)
   - 单例模式服务类
   - 会话管理
   - 数据处理
   代码量: ~450行

3. REST API层 (views.py)
   - 14个REST端点
   - 统一JSON响应
   - 完整错误处理
   代码量: ~400行

4. 数据模型层 (models.py)
   - 3个Django模型
   - 关联关系设计
   - 索引优化
   代码量: ~150行

【技术栈】

- Python: 3.12 (64位)
- Django: 5.2+
- SDK绑定: ctypes
- 数据库: SQLite
- API: REST (JSON)
- 前端: HTML5 + JavaScript
- 数据处理: numpy, Pillow

【非侵入式设计】

✅ 独立的Django应用      - apps/dm_camera/

五、API端点清单
================================================================================

【设备控制】(7个)
GET  /dm-camera/api/devices/find/     - 查找设备
POST /dm-camera/api/connect/          - 连接设备
POST /dm-camera/api/disconnect/       - 断开设备
POST /dm-camera/api/stream/start/     - 开启数据流
POST /dm-camera/api/stream/stop/      - 停止数据流
POST /dm-camera/api/capture/          - 捕获一帧
GET  /dm-camera/api/status/           - 获取状态

【配置管理】(5个)
GET    /dm-camera/api/configs/              - 列出配置
POST   /dm-camera/api/configs/create/       - 创建配置
GET    /dm-camera/api/configs/{id}/         - 获取配置
PUT    /dm-camera/api/configs/{id}/update/  - 更新配置
DELETE /dm-camera/api/configs/{id}/delete/  - 删除配置

【采集记录】(2个)
GET /dm-camera/api/captures/          - 列出记录
GET /dm-camera/api/captures/{id}/     - 获取详情

【演示页面】(1个)
GET /dm-camera/                       - Web演示界面

九、使用指南
================================================================================

【快速开始】

1. 检查集成（30秒）
   python check_dm_camera_setup.py

2. 启动服务（10秒）
   python manage.py runserver

3. 访问界面（即时）
   http://localhost:8000/dm-camera/

【开发使用】

Python代码:
   from apps.dm_camera.services import DMCameraService
   service = DMCameraService()
   service.connect()
   result = service.capture('DEPTH')

REST API:
   POST /dm-camera/api/connect/
   POST /dm-camera/api/stream/start/
   POST /dm-camera/api/capture/

JavaScript:
   fetch('/dm-camera/api/capture/', {
       method: 'POST',
       body: JSON.stringify({ frame_type: 'DEPTH' })
   })
