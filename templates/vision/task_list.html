📷 泡棉检测（2D 相机）

2D 工业相机俧诶拍摄保险杆表面白色泡棉，判定泡棉是否存在；存在则对齐、起翘均自动 OK。

  1. 进入工作台，点击「刷新预览」确认画面
  2. 首次/换型时：框选左、右泡棉 ROI 并保存标定
  3. 输入当前 POS 号，点击「拍照检测」
  4. OK 则进入下一循环；NG 则人工核查漏贴
进入泡棉检测工作台

📡 料架定位（3D 深度相机）

3D 深度相机（手眼安装）单次拍摄同时覆盖左/右双料架，输出 X/Y/Z 补偿供 PLC 与机器人修正装筱轨迹。

  1. 料架就位并锁紧，确认料框扫码和配方已完成
  2. 机器人携带 3D 相机到标准拍照位
  3. 点击「触发定位拍照」，确认左/右架置信度 ≥ 70%
  4. 补偿数据自动写入 PLC，机器人开始装筱
进入料架定位工作台
