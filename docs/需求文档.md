**2.1** **阶段一：注塑下线与激光打标（注塑机取件机器人执行）**

1. **注塑取出**：注塑机取件机器人伸入模腔，抓取保险杠产品完成产品自动下线。

2. **持件打标**：注塑机取件机器人将保险杠移动至固定式激光打标机下方预设打标位。

3. **数据交互**：PLC 发送打标触发信号至激光打标机，完成产品条码打标后（扫码枪把扫到的码信息传给PLC，视觉上位机自动读取条码信息，将条码和料框码上传至工厂 MES 系统。

4. **交接准备**：打标及数据上传完成后，注塑机取件机器人携带产品，按照预设路径移动至空中交接位，等待装箱机器人接手，同时向 PLC 发送“就绪待交接”信号。

**2.2** **阶段二：空中交接**

1. **握手确认**：装箱机器人空手移动至空中交接位，到位后向 PLC 发送“交接位就绪”信号；PLC 同时接收注塑机取件机器人的“待交接”信号与装箱机器人的“就绪”信号

2. **抓取状态确认**：PLC 下发“抓取指令”，装箱机器人启动抓手抓取保险杠，抓手真空发生器建立稳定真空后，向 PLC 反馈“真空确认”信号，证明产品已抓牢。

3. **安全释放复位**：PLC 确认装箱机器人抓牢产品后，向注塑机取件机器人下发“释放指令”；注塑机取件机器人缓慢松开工件，确认完全释放后，向 PLC 发送“释放完成”信号；随后注塑机取件机器人按照预设路径返回注塑机旁，准备抓取下一个产品，实现双机器人节拍并行。

**2.3** **阶段三：视觉装箱与泡棉组装（装箱机器人执行）**

本阶段完成视觉料架定位、产品精准装箱及泡棉贴附，全程由装箱机器人独立执行，PLC 与视觉系统协同控制，具体分为四个环节：

**2.3.1** **视觉校验与料架定位**

1. **料架就位固定**：叉车将空料架放置于指定装箱工位，操作人员确认料架摆放到位后，料架定位机构自动启动，完成料架初步固定。

2. **料框扫码**：机器人携带自动读码器到固定位置识别料框条码，传给视觉上位机获取MES配置，获取装箱的每层料架的数量和层数。

3. **机器人搭载深度相机分步拍照定位**：装箱机器人移动至左侧料架预设拍照位，利用安装于机器人手臂末端的深度相机对左侧料架进行拍照采集；视觉上位机基于采集到的深度图/点云信息，以料架刚性立柱为定位基准，计算左侧料架当前装箱区域的 X/Y/Z 三轴偏差。完成左侧料架定位后，装箱机器人再移动至右侧料架预设拍照位，对右侧料架进行同样方式的拍照采集与偏差计算，分别生成左右料架对应的补偿数据。系统调试阶段需完成深度相机与装箱机器人之间的手眼标定，确保视觉定位结果能够准确转换为机器人执行补偿数据。系统将数据发给PLC，PLC在发给机器人。

**4.**   **配置核对校验**：视觉上位机将 **3D 深度相机实测的料架层高、层距、立柱位置及当前层位空间坐标**，与 MES 系统下发的料架配方参数进行比对，确认无误后向 PLC 发送“可装箱”信号。

**2.3.2** **保险杠引导装箱**作业（根据配方数量循环作业）

PLC 将视觉系统计算的三轴偏差值下发至装箱机器人，装箱机器人根据偏差值进行伸入料架

**2.3.3** **泡棉贴附作业**（根据配方数量循环作业）

1. **泡棉抓取**：装箱机器人完成单次装箱后，移动至桶式泡棉供料台旁；供料台配备的距离传感器实时检测泡棉叠放高度，将高度数据反馈至装箱机器人，机器人调整抓手高度，利用柔性吸盘精准吸取单片泡棉。

2. **泡棉贴附**：装箱机器人携带泡棉，返回料架工位，根据预设程序调整姿态，将泡棉精准贴附于保险杠的指定防护位置，控制贴附压力均匀，避免泡棉偏移、起翘。

3. 泡棉贴附完成后，装箱机器人抓手缩回至安全位置，PLC 触发工位固定式检测相机拍照；视觉系统泡棉贴附状态进行实时校验，确认装配合格后，向 PLC 发送“工序完成”信号，本循环结束，装箱机器人等待下一个交接产品；若校验不合格，系统立即报警并锁定工位，提示人工核查处理。

 



第三阶段：B 点装箱循环阶段

本阶段包含三大核心流程：分层独立补偿、单件装箱作业、往返取件装箱循环

1. 分层独立补偿流程

   ① 每层首件触发拍照

   ② Z 轴偏差反馈补偿

2. 单件装箱作业流程

   ① 产品装入与泡棉贴附

   ② 贴附视觉校验

   ③ PLC 进度记录

3. 往返取件装箱循环（整体循环调度流程，承载单件装箱作业持续执行）

一般情况下，有两个料架并在一起，每个料架有3层。有2个2D相机，1个3D深度相机。1号2D相机拍左边的料架，2号2D相机拍右边的料架。1号2D相机拍左边的料架，根据层高把一张图片分成每层的部分，假如有3层料架，把一张图片分别上中下三部分；2号料架同理。1个3D深度相机根据PLC的指令在某处进行拍照，根据料架每层的首个进行拍照，循环如此。





双料架 3 层保险杠视觉装箱系统技术实现说明
一、系统核心逻辑

本系统用于保险杠产品在双料架、三层结构中的自动装箱作业。系统由装箱机器人、PLC、视觉上位机、3D 深度相机、2D 工业相机、料架定位机构、泡棉供料装置及 MES 系统组成。

其中，1 台 3D 深度相机安装于装箱机器人末端，用于料架空间位置识别、层位定位及装箱补偿；2 台 2D 工业相机分别对应左、右料架，用于检测料架内泡棉是否贴附到位。装箱机器人负责产品接收、装箱、泡棉抓取及料架内泡棉贴附动作。PLC 负责整体节拍控制、信号交互、进度记录及异常报警。

本项目中的泡棉贴附对象为料架，而非保险杠产品。泡棉应贴附在料架内部指定的支撑面、隔板、侧挡或产品接触位置，用于防止保险杠在装箱、存放及转运过程中与料架结构直接接触，造成划伤、压痕或变形。

二、设备分工说明
2.1 3D 深度相机

3D 深度相机安装于装箱机器人第六轴末端，与机器人抓手集成布置。机器人根据 PLC 指令携带 3D 深度相机移动至当前料架、当前层位的标准拍照位置，对料架层板、立柱、支撑面或定位基准区域进行拍照。

视觉上位机根据深度图或点云数据，计算当前料架、当前层位的实际空间偏差，输出 Z 轴高度补偿值，必要时可同步输出 X/Y/Z 三轴补偿数据。补偿结果写入 PLC 数据区，由 PLC 下发给装箱机器人，用于修正装箱轨迹和泡棉贴附轨迹。

2.2 2D 工业相机

系统配置 2 台 2D 工业相机，1 号相机对应左料架，2 号相机对应右料架。2D 相机不用于料架整体分层识别，也不通过拍摄整张料架图片后切割为上、中、下三层进行检测。

2D 相机的主要作用是检测当前工位、当前层位、当前贴附区域的泡棉是否贴附合格，包括是否漏贴、位置是否偏移、是否起翘、是否覆盖到指定区域等。

每次泡棉贴附完成后，装箱机器人将对应贴附区域移动至 2D 相机视野范围内，或由固定相机直接拍摄当前贴附区域，PLC 触发相机拍照，视觉上位机完成图像分析并向 PLC 输出 OK/NG 结果。

2.3 装箱机器人

装箱机器人主要完成以下动作：

从空中交接位接收注塑机取件机器人传递的保险杠产品；
根据 PLC 下发的目标料架、目标层号、目标位置号及视觉补偿数据，将保险杠装入料架；
移动至泡棉供料台抓取单片泡棉；
返回料架内部，将泡棉贴附在料架指定位置；
将贴附区域移动至检测位置，配合 2D 相机完成泡棉贴附质量检测；
检测合格后返回空中交接位，等待下一件产品。
三、标准作业流程
3.1 系统初始化
料架就位
叉车将左、右两个空料架放置于指定装箱工位，人工确认料架位置基本到位。
料架定位与锁紧
PLC 控制料架定位机构动作，对左右料架进行机械限位、夹紧或固定，确保料架在装箱过程中不发生明显位移。
料框扫码
装箱机器人携带扫码器移动至料框条码位置，分别读取左、右料架条码。扫码结果写入 PLC 数据区，视觉上位机读取后向 MES 系统请求对应装箱配方。
MES 配方获取
视觉上位机根据料框条码获取料架配方，包括料架层数、每层装箱数量、层高参数、装箱顺序、泡棉贴附位置、检测区域等工艺参数。
初始 3D 定位
机器人携带 3D 深度相机分别移动至左、右料架第一层标准拍照位，采集料架当前层位数据。视觉系统计算左右料架第一层补偿值，并写入 PLC 数据区。
可装箱确认
PLC 确认料架扫码完成、配方获取完成、料架定位完成、第一层视觉补偿数据有效后，向装箱机器人发送“可装箱”信号。
3.2 单件装箱流程
目标位置分配
PLC 根据当前生产进度、MES 配方和装箱顺序，判断当前保险杠应装入左料架或右料架，并确定对应层号和位置号。
空中交接取件
装箱机器人移动至空中交接位，与注塑机取件机器人完成产品交接。真空确认或夹持确认完成后，PLC 允许注塑机取件机器人释放产品。
产品装入料架
装箱机器人调用当前料架、当前层位的视觉补偿数据，将保险杠按照预设轨迹装入料架指定位置。装箱过程中机器人应根据料架空间、产品姿态及补偿数据控制进入角度和插入深度，避免产品与料架发生碰撞。
装箱完成反馈
保险杠装入到位后，装箱机器人向 PLC 反馈“产品装箱完成”信号。PLC 记录当前料架侧别、层号、位置号和产品条码信息。
3.3 料架内泡棉贴附流程
泡棉抓取
产品装箱完成后，装箱机器人移动至泡棉供料台。供料台通过距离传感器检测泡棉叠放高度，并将高度数据反馈给机器人或 PLC。机器人根据泡棉高度修正抓取位置，利用柔性吸盘吸取单片泡棉。
进入料架贴附位置
机器人携带泡棉返回当前目标料架，根据当前层号、位置号及视觉补偿数据，移动至料架内部指定贴附位置。贴附位置为料架内部与保险杠接触或可能接触的防护区域，包括层板、支撑面、侧挡、隔板或其他指定防护点。
泡棉贴附到料架
机器人按照预设姿态和压力，将泡棉贴附在料架指定位置。贴附过程中应保证泡棉与料架表面充分接触，位置准确，不得出现漏贴、歪斜、起翘、未压实等问题。
贴附完成反馈
泡棉贴附完成后，装箱机器人向 PLC 反馈“泡棉贴附完成”信号，并退出至安全检测位置，等待 2D 视觉检测。
3.4 2D 泡棉贴附质量检测流程
检测触发
PLC 在收到机器人“泡棉贴附完成”信号后，触发对应料架的 2D 工业相机拍照。左料架由 1 号 2D 相机检测，右料架由 2 号 2D 相机检测。
检测对象
2D 相机检测对象为料架内部当前刚贴附完成的泡棉区域，而不是保险杠表面。视觉系统根据当前料架侧别、层号、位置号调用对应检测模板和 ROI 区域。
检测内容
视觉上位机对图像进行处理，判断泡棉是否存在以下问题：
泡棉漏贴；
泡棉位置偏移；
泡棉角度异常；
泡棉起翘；
泡棉覆盖区域不足；
泡棉未贴到指定料架位置。
检测结果反馈
检测合格时，视觉上位机向 PLC 发送“泡棉检测 OK”信号，PLC 记录当前产品装箱及泡棉贴附完成状态，并允许机器人进入下一循环。

检测不合格时，视觉上位机向 PLC 发送“泡棉检测 NG”信号，同时输出缺陷类型和对应料架位置。PLC 触发报警并锁定当前工位，提示人工核查处理。人工确认后，系统可根据现场策略选择重新贴附、人工补贴或跳过处理。

3.5 分层补偿与循环作业
每层首件补偿
左、右料架每一层开始装箱前，PLC 触发机器人携带 3D 深度相机移动至该层标准拍照位，采集当前层位信息。视觉系统计算该层补偿值后写入 PLC 数据区。
同层数据复用
同一料架、同一层内的后续产品装箱和泡棉贴附，优先复用该层首件前采集的补偿数据。若现场要求更高精度，也可设置每隔固定件数重新拍照校正一次。
左右料架独立管理
左、右料架的料框条码、当前层号、当前位置号、补偿数据、装箱数量、泡棉贴附结果和异常状态应独立存储，不得混用。
层满切换
当某一料架当前层达到 MES 配方设定装箱数量后，PLC 将该层状态置为“层满”，并触发机器人对下一层进行 3D 拍照定位。新一层补偿数据生成后，系统进入下一层装箱及泡棉贴附流程。
满架处理
当某个料架三层全部装满后，PLC 发出“满架”提示信号，并停止向该料架继续装箱。操作人员解除定位机构，叉车将满架转运离开，并放置新的空料架。新料架就位后，系统重新执行料框扫码、配方获取、3D 定位及装箱循环流程。
